# AI工程师实习生笔试题

## 实习生笔试题：中文口播文本转英文翻译+语音小工具

### 我们考察什么
* 能否独立将一个真实需求快速变成可运行的工具
* 遇到不熟悉的API和技术，能否自己找到解决方案
* 工具是否真的能用，而不只是能跑
* 是否具备 AI Native 的工作方式，善用AI辅助完成开发

### 背景
这是我们日常业务中真实使用的场景：运营人员输入中文口播文案，需要快速生成英文翻译，并生成可直接使用的英文语音，用于短视频和直播投放。本题来自我们的真实业务场景，已精简为半小时至一小时内可完成的版本。

### 你需要做什么
实现一个可以运行的网页小工具，具备以下功能：

#### 输入
* 文本输入框，支持输入中文口播文本（示例文本见下方）

#### 输出
* 展示翻译后的英文文本，支持一键复制
* 提供英文音频的播放和下载按钮

### 技术提示
* 语音生成使用 ElevenLabs API（自行注册，有免费额度）

### 进阶挑战（可选，额外加分）
如果你觉得基础功能太简单，可以选择以下任意一项或多项继续完善，我们会额外加分：
* 翻译结果可以针对短视频场景做优化，比如控制句子长度、更口语化
* 历史记录功能，保存之前生成过的文本和音频
* 界面支持批量输入，一次处理多条文案
* 其他你觉得可以做的部分

> **注：** 进阶部分没有标准答案，我们更想看你怎么思考“让这个工具更好用”这件事。
